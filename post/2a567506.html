NRM的出现原因

由于 NPM 官方默认的包注册源位于国外，直接从该源下载包可能速度较慢，许多开发者会选择使用如淘宝 NPM 镜像这样的国内镜像源。nrm 就是用来帮助用户在不同镜像源之间快速切换的工具。

nrm作用

nrm主要用来便捷地管理和切换不同的 NPM (Node Package Manager) 包注册源，这对于提高依赖包的下载速度非常有用，特别是对于那些在中国或其他国际网络访问受限地区的开发者。
