1、问题描述

  • 之前碰到过此类问题，原因是之前用了测试版本的pod，然后回归正式版本，导致找不到pod文件

2、解决方案

  • 2.1 方案一：
    • 收先关闭xcode 然后pod install
    • 如果不行就先删除xcworkspace Podfile.lock Pods文件夹 ~/Library/Developer/Xcode/DerivedData路径下对应工程的文件夹
    • 然后 pod install
  • 2.2 方案二：
    • pod deintegrate –verbose
    • pod install –verbose
