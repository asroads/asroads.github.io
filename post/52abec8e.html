安装 Homebrew

**小提示:**如果想卸载Homebrew需要(其实就是把“install”改成“uninstall”)

安装软件，如：brew install oclint

卸载软件，如：brew uninstall oclint

搜索软件，如：brew search oclint

更新软件，如：brew upgrade oclint

查看安装列表， 如：brew list

更新Homebrew，如：brew update
