生成的json文件

  • input/TrackData.xlsx 文件路径
  • -s Sheet1 表名
  • -key id 主键

json文件转换成Tyepscript

  • ./input/TrackData.json // json文件路径
  • --output ./output //输出目录 ./output
  • --enumName TrackType //生成的枚举名字 TrackType
  • --fileName TrackFile // 生成的Typescript文件名字 TrackFile
  • --obj //生成对应的 对象文件 如果不写就不生成
